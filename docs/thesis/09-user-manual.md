# 09 - User Manual

## Dang Ky Va Dang Nhap

1. Truy cap trang web.
2. Chon Dang ky.
3. Nhap ho ten, email, so dien thoai, mat khau.
4. Nhap OTP de xac thuc email.
5. Dang nhap bang email va mat khau.

## Dat Lich Kham

1. Chon Dat lich kham.
2. Chon co so, chuyen khoa, bac si.
3. Chon ngay va khung gio con trong.
4. Nhap ly do kham.
5. Kiem tra thong tin BHYT neu su dung.
6. Xac nhan dat lich.

## Quan Ly Lich Hen

1. Vao Lich hen cua toi.
2. Xem lich sap toi, hom nay va lich su.
3. Mo Xem chi tiet de xem hanh trinh lich hen.
4. Tai phieu dat lich hoac phieu kham neu co.
5. Gui yeu cau huy/doi lich neu he thong cho phep.

## Ho So Kham Benh

1. Vao Ho so kham benh.
2. Loc theo trang thai ho so hoac nam.
3. Mo chi tiet ho so.
4. Xem chan doan, don thuoc, file dinh kem, tai kham.
5. Tai PDF ket qua kham.

## Danh Gia Bac Si

1. Chon lich da hoan thanh.
2. Chon Danh gia bac si.
3. Nhap so sao va nhan xet.
4. Gui danh gia.

## Luu Y

- Den truoc gio kham khoang 15 phut.
- Mang CCCD hoac giay to tuy than.
- Mang the BHYT neu co su dung.
- Mang ho so kham cu neu co.
